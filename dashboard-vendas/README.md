# Dashboard de Vendas Globais

# Objetivo 
Acompanhar o desempenho de vendas da empresa analisando o número de vendas, quantidade de vendas por categoria, quantidade de vendas realizadas por país, a média de descontos por subcategorias e quais países tiveram a maior média de vendas.

# Fontes de Dados
Base de Vendas

# Visualizações 
- Cartões KPI
- Gráfico de pizza
- Gráfico de barras clusterizado
- Gráfico de barras empilhadas
- Mapa
- Segmentadores
